# NexGen Exam Portal (Premium SaaS Edition)

A commercial‑grade online examination platform for educational institutions with separate Admin and Student portals.

## Features

- **Admin Portal**: Manage students, tests, questions (bulk CSV upload), monitor live tests, pause/resume tests, community moderation, messaging.
- **Student Portal**: Take tests with timer, anti‑cheating (tab switch detection), pause overlay, view results with analysis, class community chat, direct messaging with admin.
- **Security**: Environment‑based credentials, token‑protected admin routes.
- **Tech Stack**: HTML5, Tailwind CSS, Vanilla JS, Node.js/Express, MongoDB Atlas, Vercel deployment.
